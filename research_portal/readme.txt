# Research Portal Management System

## Overview
A web-based system for managing research submissions and student-faculty collaboration.

## Features
- User registration and login (student/faculty roles).
- Thesis submission and viewing for students.
- Dashboard and student management for faculty.
- Profile management for all users.

## Installation
1. Clone the repository.
2. Import `research_portal.sql` into your MySQL database.
3. Configure `includes/config.php` with your database credentials.
4. Run the project on a local server (e.g., XAMPP or WAMP).
